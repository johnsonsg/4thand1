# Onboarding

## Prerequisites

- Node.js (current LTS recommended)
- npm

## Install

From repo root:

- `npm install`

## Run

From repo root:

- `npm run dev`

Then open http://localhost:3000

## Project Layout

- `frontend/` — Next.js app (App Router) + Payload integration
- `Docs/` — project docs and how‑to guides

## Useful Commands (root)

- `npm run dev` — start dev server (frontend workspace)
- `npm run build` — build frontend
- `npm run lint` — lint frontend

## CMS Modes

The frontend supports a mock CMS by default and can be pointed to a real backend later. See frontend README:

- [frontend/README.md](../frontend/README.md)
